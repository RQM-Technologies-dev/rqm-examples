# AGENTS.md — AI Contributor Rules for `rqm-examples`

`rqm-examples` is a demonstration repository.

Never implement mathematical primitives here.

All math must come from:
- rqm-core
- rqm-qiskit

This repo exists only to show usage of those libraries.

## Rules

- Do not add quaternion, spinor, Bloch, or SU(2) implementations.
- Do not reimplement any functionality already provided by `rqm-core` or `rqm-qiskit`.
- Examples must remain simple, readable, and educational.
- Each script should demonstrate exactly one concept.
- All scripts must print human-readable results.
